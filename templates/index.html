Bitiruv / kurs loyihasi

Dars jarayonida tinglovchilar faolligini avtomatik baholash tizimi

Loyiha Zoom darsi yoki kompyuter ekranidagi jonli oqim orqali tinglovchilar mavjudligi, diqqat holati, harakat darajasi va ehtimoliy hodisalarni real vaqt rejimida tahlil qiladi. Bosh sahifa faqat tanishtiruv uchun; barcha sozlamalar admin panelda, kuzatuv esa alohida hodisa sahifasida bajariladi.

Hodisa kuzatuvini ochish Admin panel
Saqlangan sessiya {{ camera_settings.room_name or 'Zoom darsi' }}
Zoom manbasi {% if camera_settings.camera_source %} {{ camera_settings.camera_source }} {% else %} Virtual kamera indeksi: {{ camera_settings.camera_index }} {% endif %}
Tizim shaxsni tanimaydi. Natijalar anonim tracklar va ehtimoliy hodisa signallari sifatida ko'rsatiladi.

Loyiha vazifasi

Zoom oynasi yoki virtual kamera orqali darsdagi umumiy faollikni avtomatik baholash, o'qituvchiga real vaqt indikatorlari va sessiya yakunida saqlangan statistikani taqdim etish.

Hodisa kuzatuvi nima qiladi?

Hodisa kuzatuvi jonli video, ballar, talabalar tracklari, telefon ehtimoli, savol ishorasi, uyquchanlik va chalg'ish kabi barcha monitoring ko'rsatkichlarini bitta sahifada ko'rsatadi.

Admin vazifasi

Admin panelda Zoom ekran manbasi yoki virtual kamera tanlanadi va talabalar bazasi jadval ko'rinishida yuritiladi.
