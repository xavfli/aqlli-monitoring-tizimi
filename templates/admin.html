Yo'riqnoma

1. Kamerani tarmoqqa ulang

Universitet IP kamerasi kompyuter bilan bir xil lokal tarmoqda bo'lishi kerak. Kamera IP manzilini router, NVR yoki kamera admin sahifasidan oling.

2. RTSP manzilini kiriting

Ko'p kameralar uchun format: rtsp://login:parol@IP:554/stream1. Masalan: rtsp://admin:12345@192.168.1.25:554/stream1.

3. Default kamerani sinang

Agar hozircha universitet kamerasi manzili bo'lmasa, Webcam 0 yoki Ochiq RTSP test variantini tanlab saqlang.

4. Xonani nomlang

E 102 xona kabi nom kiriting. Bu nom bosh sahifada sessiya nomi sifatida chiqadi va saqlangan kamera bilan birga qoladi.

5. Monitoringni boshlang

Saqlagandan keyin bosh sahifaga qayting. RTSP maydoni to'lgan bo'lsa, tizim universitet kamerasini ishlatadi. Bo'sh bo'lsa, webcam indekslari sinab ko'riladi.
