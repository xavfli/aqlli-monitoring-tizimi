# Taqdimot Uchun Qisqa Matn

## Loyihaning kuchli tomonlari

- Python asosida ishlab chiqilgan.
- Computer Vision texnologiyasi qo'llangan.
- Kamera integratsiyasi mavjud.
- Real-time monitoringni amalga oshiradi.
- O'qituvchi uchun web dashboard tayyorlangan.
- Natijalar avtomatik ravishda SQLite bazasiga saqlanadi.
- Sessiyalarni CSV formatga eksport qilish mumkin.

## Himoyada aytish mumkin bo'lgan gap

Ushbu loyiha dars jarayonida tinglovchining faolligini avtomatik baholashga qaratilgan. Tizim kameradan real vaqt rejimida video oqim oladi, yuzni aniqlaydi, tinglovchining mavjudligi, diqqat darajasi va harakatini baholaydi. Shundan so'ng dashboardda statistik ma'lumotlarni ko'rsatadi va sessiya tugagach natijalarni ma'lumotlar bazasiga saqlaydi. Bu yechim inson omilini kamaytiradi va baholash jarayonini ancha tezlashtiradi.
